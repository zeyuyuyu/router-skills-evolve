# 35B Joint Evolution Results

This directory contains the public result artifacts for the completed 35B
skills + LLM + router joint-evolution run.

Included:

- `final_ablation_table.md`: final aggregate table across 4 cycles.
- `curve.png`: per-cycle aggregate curve.
- `cycle_*/e2e_ablation_summary.json`: per-cycle E2E ablation metrics.
- `cycle_*/e2e_ablation_summary.md`: per-cycle E2E ablation table.

Large checkpoints, raw traces, service logs, and machine-specific runtime files
are intentionally not committed.
