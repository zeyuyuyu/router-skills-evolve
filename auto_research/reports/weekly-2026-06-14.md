## Headline Findings

### 1. Router Underperforms Base on Tau2 Held-Out (CRITICAL NEGATIVE RESULT)

`cwy_35b_fullsplit_20260610_083624` established the first proper 178/100 train/test
split for tau2 agentic routing. The held-out evaluation (100 tasks, 80% easy / 20% hard)
shows:

- **Base (always-small): 80% task pass, 10% cost**
- **Router (TF-IDF+LR): 71% task pass, 46% cost** ← WORSE than base
- **Large (SFT 35B always): 71% task pass, 100% cost** ← also WORSE than base

The router *actively harms* task performance on held-out: −9pp task pass, +36pp cost.

**Root cause**: The SFT-trained 35B model underperforms the base small model on autonomous
task completion (as established by the 0% Method A result from earlier). When the router
sends easy tasks to this inferior student, those tasks fail. The router compounds an
underlying model quality problem rather than solving it.

**Secondary finding**: TF-IDF+LR router generalizes poorly — routing accuracy drops 18pp
from training split (84.3%) to held-out (66.0%), suggesting vocabulary overfitting.

**Reactive queuing**: EXP-066 (L2 regularization sweep, priority 9, CPU-only) and
EXP-067 (tau2 agentic GRPO, priority 8, 8× H200) were queued the same day in response.

### 2. In-Distribution Joint Cycle Shows Strong Router Performance

`cwy_35b_joint_20260606_165203` (4 cycles, n=74, no train/test split) shows the router
achieving 82–93% routing accuracy vs 57–85% base across cycles. This confirms the router
works well in-distribution but overstates generalization ability (which §5.4.2 corrects).

### 3. Curriculum Learning Track Opened

EXP-062/063 (RECRL easy-to-hard, 200 and 400 tasks) open the `grpo_curriculum_continual`
kind for the first time across all 69 queued experiments. Motivated by arxiv:2605.00433
(+2.1pp MBPP on Qwen2.5-Coder-1.5B) and directly targeting the entropy-collapse hypothesis
for the 200→400 task degradation cliff.

### 4. Combined Advantage Fix + Rollout Diversity (EXP-064)

EXP-064 combines two previously queued single-factor experiments (Dr.GRPO advantage
debiasing, EXP-060; T=1.2 rollout diversity, EXP-061) to test additivity. If both fixes
compound, the +2pp ceiling could reach +3–4pp without algorithm change.

### 5. EP-GRPO: Entropy-Gated Zero-Variance Fix (EXP-068)

EP-GRPO (arxiv:2605.04960) directly targets the 73%-silent-group problem via token-level
entropy gating and implicit process signal from policy/reference log-prob ratio. Reported
+4.79pp on Qwen2.5-3B math; EXP-068 is the first test on code generation. Implementation
overhead is zero (both signals already computed in the GRPO training loop).
